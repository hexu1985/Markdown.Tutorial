### 数学公式编号

在Markdown中使用LaTeX公式时，加上`\tag{...}`标签可以生成对应的编号。

```
比如这个代码可以生成
$$
x + y = z     \tag{1.1}
$$
这样的带编号的公式。
```

---

也可以使用LaTeX标准的`\begin{equation}`和`\end{equation}`自动编号。
因为自动编号的代码较为复杂，而且不易扩展，所以不太建议使用自动编号，手动编号更易维护。

公式引用时候可以直接用`$编号$`即可。

```
对于
$$
a^2+b^2=c^2     \tag {1.2}
$$
由公式$(1.2)$即可得到结论。
```
