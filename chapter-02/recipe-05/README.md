### 代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

```
`printf()` 函数
```

显示效果如下所示：

`printf()` 函数


#### 代码区块

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：

<pre><code>
```javascript
$(document).ready(function () {
            alert('RUNOOB');
            });
```
</code></pre>


显示效果如下所示：

```javascript
$(document).ready(function () {
            alert('RUNOOB');
            });
```
